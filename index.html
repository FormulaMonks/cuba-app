Cuba App

This is a sample application using Cuba and Mote.

Getting started

Clone this repository and install the gems listed in .gems. If you want to make your life easier, run gem install dep first, then dep install.

Testing

Tests are located in /test and they use Cutest and Capybara. You will find a very basic example that checks the homepage for certain content.

More information

Meet us on IRC at #cuba.rb on freenode.net.
